== Description ==

Seentient Related Products Widget automatically finds and displays apparel that is relevant to the content on your blog or e-commerce site.

= Features =

* Monetize your blog posts
* Uses machine learning to find simlar and exact matches for the apparel depicted in your images.
* Show on every page after content
* Show on sidebar widget on every page

= Optimized =

* Asynchronous script will not slow or block your website pages from loading.
* World-wide CDN ensures fast content delivery for all of your visitors.
* Mobile devices detection and compatibility.

== Installation ==

= Automatic installation through WordPress =

1. Log-in to your WordPress site.
2. Hover over Plugins and click Add New.
3. Under Search type in "seentient related product", then click Search Plugins.
4. In the results page, click Install Now.
5. Once installed, click Activate Plugin. You're done!

= Manual installation =

1. Download fresh version here [https://wordpress.org/plugins/seentient/](https://wordpress.org/plugins/seentient/) .
2. Unzip the Seentient.*.*.zip file. 
3. Upload the Seentient folder (not just the files in it!) in your /wp-content/plugins folder. If you're using FTP, use 'binary' mode.
4. In your WordPress administration, go to the Plugins page.
5. Activate the Seentient Related Products plugin and a subpage for Seentient Related Products will appear in your Options menu.

To upload the plugin through WordPress, instead of FTP - upload the downloaded zip file on the 'Add New' plugins screen in your WordPress admin area and activate.
